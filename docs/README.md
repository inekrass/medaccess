# Документация проекта MedAccess

Папка `docs` содержит вспомогательные материалы для учебного UI/UX-проекта MedAccess. Основная часть документации посвящена PlantUML-диаграммам, которые описывают пользовательский сценарий и структуру интерфейса.

## PlantUML-диаграммы

Диаграммы находятся в папке `docs/plantuml`.

| Файл | Назначение |
| --- | --- |
| `user-behavior.puml` | Описывает основной сценарий поведения пользователя при записи к врачу. |
| `interface-structure.puml` | Показывает связь основных экранов и панели персонализации. |

## Use cases

Use cases используются для описания действий пользователя в основном сценарии записи к врачу. В проекте они представлены в диаграмме `user-behavior.puml`.

| Код | Название | Результат |
| --- | --- | --- |
| UC1 | Открыть главную страницу | Пользователь видит стартовый экран сервиса. |
| UC2 | Настроить параметры отображения | Интерфейс адаптируется под выбранные параметры. |
| UC3 | Перейти к записи к врачу | Пользователь начинает сценарий записи. |
| UC4 | Выбрать специалиста | Пользователь сужает список доступных врачей. |
| UC5 | Выбрать врача | Система фиксирует выбранного врача. |
| UC6 | Выбрать дату и время | Система фиксирует выбранный слот приема. |
| UC7 | Заполнить данные пациента | Пользователь вводит данные для записи. |
| UC8 | Исправить ошибки формы | Пользователь получает текстовые ошибки и корректирует поля. |
| UC9 | Подтвердить запись | Пользователь проверяет данные и подтверждает действие. |
| UC10 | Получить подтверждение записи | Пользователь видит итоговое сообщение об успешной записи. |
